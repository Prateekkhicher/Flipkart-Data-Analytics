# Raw Data

This folder contains the original Flipkart e-commerce datasets used in this project.

> Note: The datasets are not included in this repository due to GitHub file size limitations.

The data can be downloaded from Kaggle and placed in this folder before running the notebooks.

Expected files:

- customers.csv
- orders.csv
- order_items.csv
- products.csv
- sellers.csv
- payments.csv
- reviews.csv
- shipmenta.csv
- returns.csv
